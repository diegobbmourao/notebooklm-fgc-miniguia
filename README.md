# notebooklm-fgc-miniguia
Este projeto tem como objetivo estruturar um miniguia introdutório sobre o Fundo Garantidor de Créditos (FGC), utilizando o NotebookLM como ferramenta de aprendizagem ativa. A proposta integra curadoria normativa, elaboração de perguntas estratégicas e testes de prompts para consolidar conhecimento regulatório.
